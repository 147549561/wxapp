# wxapp
weui-wxss中的样式还算齐全，不过根据自己平时的业务需要又新增了一些组件，目前想到的暂时为以下几个：
> 1. alert提示（错误，警告，成功，加载，icon可选，文字可配置）
> 2. 弹出层（全屏，下半屏）（常用于选择商品属性）
> 3. icheck单选按钮，复选按钮，on/off开关
> 4. 加载样式，无数据提示等tip
> 5. 文字跑马灯
> 6. 富文本解析
> 7. 横向滚动图文
> 8. 左侧tab
> 9. 左滑取消操作[可选]、删除
> 10. 全选/全不选
> 11. 视频全屏播放
> 12. 省市区三级联动（单页面（小程序的页面层级限制）、多页面两种）
> 13. 瀑布流图文（商品列表，宽度固定，高度自适应）
> 14. 3D图片轮换
> 15. 横向可滑动tabbar
